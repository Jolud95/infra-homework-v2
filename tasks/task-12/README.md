# Docker container

Необходимо сделать образ и запустить докер контейнер внутри которого будет запущена нода с `server.js`.

Контейнер должен запуститься на 3000 порту.

Сам сервер при получении запроса ходит в pokeAPI и возвращает информацию о покемоне.

Изучите код server.js, он ожидает что у него будет доступен файл с секретами внутри контейнера по пути `/var/static/secrets.js`.

Для тестирования используйте `yarn test`.

## Примечания

Проброс файла с секретами должен быть выполнен с помощью bind mount.

Не забудьте пробросить порт.
